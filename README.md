## Dataset Structure
Raw images are stored in `data/raw/<GroupMemberName>/<SubjectName>/`

Processed imagesa are stored in `data/processed/SubjectName>/` 

Group member names are included in raw data for transparency and ease of tracking individual contributions.
Group member names are removed during processing to prevent unnecessarily complicating the dataset.
